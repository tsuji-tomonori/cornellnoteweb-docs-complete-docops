# DD-UI-006 アクセシビリティ検証（チェックリスト/テスト）

## チェックリスト（抜粋）
- [ ] 主要操作がキーボードのみで可能（Tab/Shift+Tab/Enter/Esc）
- [ ] モーダルのフォーカストラップが動作する
- [ ] `aria-live` で保存状態が通知される
- [ ] ラベルと入力が紐づいている（`for`/`id`）
- [ ] 色だけに依存しない（保存状態のテキストあり）

## 手動テスト手順（例）
1. /notes で検索入力→一覧が絞り込まれる（キーボード操作のみ）
2. /editor でモーダルを開く→Tabがモーダル内に閉じる→Escで閉じる
3. 保存状態の読み上げ確認（スクリーンリーダー環境）
