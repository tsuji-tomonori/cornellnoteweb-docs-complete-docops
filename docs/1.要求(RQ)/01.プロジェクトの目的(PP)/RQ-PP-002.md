# RQ-PP-002 ノートの再発見と復習を容易にする

## SnowCard（日本語）
- **要求ID**: RQ-PP-002
- **種別**: プロジェクトの目的
- **優先度**: Should
- **要求**: 本プロジェクトは、利用者が過去のノートを分類・検索により迅速に再発見し、復習に利用できる状態を実現する。
- **根拠**: ノート数が増えるほど価値が増すため、整理（ノートブック/タグ）と検索は継続利用の鍵となる。
- **成功条件（Fit Criterion）**:
  - ノート100件の状態で、指定ノートに到達するまでの所要時間が平均30秒以内となる（シナリオテスト）。
