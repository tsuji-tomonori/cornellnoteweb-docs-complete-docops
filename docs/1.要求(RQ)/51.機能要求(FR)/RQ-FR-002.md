## SnowCard（日本語）
- **要求ID**: RQ-FR-002
- **種別**: 機能要求
- **優先度**: MUST
- **要求**: 本システムは、/api/auth/login と /api/auth/logout によりログイン状態を管理できる。
- **根拠**: ノートの保護と個人別データのため。
- **受入基準**:
  - ログイン成功で /notes が閲覧できる
  - ログアウトでローカルセッションが消える
- **例外/エラー**:
  - 失敗→401相当（将来）
- **関連**:
  - [[RQ-UC-001]]
  - [[RQ-SEC-001]]
  - [[DD-API-002]]

## テスト観点（抜粋）
- 正常系/異常系（パラメータ不正、存在しないID、未ログイン）
- 権限（将来）: 他ユーザーのノートにアクセスできない
- NFR: レイテンシ/監査ログ（[[RQ-PS-001]] / [[RQ-OBY-001]]）
