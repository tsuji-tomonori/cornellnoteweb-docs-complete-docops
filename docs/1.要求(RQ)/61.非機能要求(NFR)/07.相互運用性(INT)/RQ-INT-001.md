## SnowCard（日本語）
- **要求ID**: RQ-INT-001
- **種別**: 非機能要求（Non-functional（Interoperability））
- **優先度**: SHOULD
- **要求**: 本システムは、外部システム連携において、標準化されたプロトコルおよび認証方式（例：HTTPS、OIDC/OAuth2等）を使用しなければならない。
- **根拠**: 連携容易性と保守性を確保するため。
- **受入基準**:
  - インタフェース仕様書にプロトコル・認証方式が明記され、標準外方式は例外承認を要することを確認する（検査）。
- **例外/エラー**:
  - 前提条件や依存要件が未達の場合は、暫定対策・期限・責任者を記録し、次回リリースまでに是正する。
- **依存・関連**:
  - なし
- **起案者**: アーキテクト
- **参考資料**: API標準、セキュリティ標準
